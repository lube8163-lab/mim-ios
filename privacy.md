Privacy Policy for Mim

Mim is a beta social application that allows users to share images and interact with generated content.

- Content uploaded by the user (images and related metadata)

How We Use Information

- To improve stability and user experience
- To communicate with users regarding the app, if necessary

Data Storage and Security
- Data is transmitted using standard secure communication protocols (HTTPS)

Third-Party Services
- Cloud infrastructure services may be used to store and process data
- Apple-provided system services are used where applicable

Tracking and Advertising
- Mim does not use data for tracking users across apps or websites
- Mim does not display third-party advertisements or sell user data

User Control
- Users may request deletion of their account and associated data

Changes
